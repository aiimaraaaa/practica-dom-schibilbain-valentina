# Galería de Superhéroes

Práctica de manipulación del DOM con JavaScript.

## Descripción

Aplicación web que permite visualizar, agregar, eliminar y filtrar personajes de una galería de superhéroes, generados dinámicamente desde JavaScript.

## Funcionalidades

-Renderizado de cards desde un arreglo de objetos.
-Filtro de personajes por nombre.
-Formulación para agregar nuevos personajes.
-Eliminación de personajes de la página.

# Tecnologías utilizadas

-HTML
-CSS
-JavaScript
-Bootstrap 5

# Flujo de trabajo Git

- `main` -> estructura base del proyecto
- `feature/catalogo` -> renderizado de cards y filtro
- `feature/formulario` -> formulario y eliminación
